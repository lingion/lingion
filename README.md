# lingion

哈尔滨工程大学在读。

课表 app 有广告、AI 客户端移动端体验差、给 AI agent 画图的工具不够灵活——于是自己写了。

## 仓库

- [AIaW](https://github.com/lingion/AIaW) (fork · 223 commits ahead) — AI 对话客户端原生移动端：iOS Xcode 项目、图片手势缩放、离线导出管线、Cerebras + MiniMax provider 接入
- [plot-mcp-worker](https://github.com/lingion/plot-mcp-worker) — MCP 图表引擎跑在 Cloudflare Workers 上；CJK 字体文本转路径管线；40+ 可视化工具
- [sleepy](https://github.com/lingion/sleepy) — 21 个 release 的 Android 课表 app；支持 9 种教务协议直连导入、Material You 主题、桌面小组件
- [wakeup-mod](https://github.com/lingion/wakeup-mod) — WakeUp 课程表逆向工程；剥离 40,000+ SDK 类，纯净 24MB APK
- [local-policy-agent](https://github.com/lingion/local-policy-agent) — 便携式本地策略代理运行时
- [HEU-keep](https://github.com/lingion/HEU-keep) — Keep 运动数据生成器
- [qdp](https://github.com/lingion/qdp) — Qobuz 下载工具
- [Dell-Precision-7740-OpenCore-EFI](https://github.com/lingion/Dell-Precision-7740-OpenCore-EFI) — Dell Precision 7740 黑苹果 OpenCore EFI

## 技术栈

Android (Kotlin / Jetpack Compose · Material You) · TypeScript (Cloudflare Workers · MCP) · Python · APK 逆向

偶尔在 [blog.qdp.qzz.io](https://blog.qdp.qzz.io) 写东西。
